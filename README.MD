## âš™ï¸ Setup

1. Clone repositori:
   ```bash
   git clone https://github.com/Fadlihardiyanto/football-api.git
   cd football-api
   ```

2. Copy file `.env.example` ke `.env` dan sesuaikan konfigurasi:
   ```bash
   cp .env.example .env
   ```

3. Unduh dependensi:
   ```bash
   go mod tidy
   ```

4. Build project:
   ```bash
   make build
   ```

5. Jalankan migrasi database:
   ```bash
   make migrate-up
   ```
